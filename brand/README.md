# BSD Lab Brand Starter

This folder contains the shadcn-inspired revision of the **Brain State Dynamics Lab** logo system.

## Deliverables

- `assets/logo/bsd-mark.svg`: app-icon style mark with rounded container on light backgrounds
- `assets/logo/bsd-mark-dark.svg`: app-icon style mark for dark surfaces
- `assets/logo/bsd-mark-mono.svg`: monochrome container mark for print and simple headers
- `assets/logo/bsd-mark-raw.svg`: pure symbol without the container
- `assets/logo/bsd-mark-raw-dark.svg`: pure symbol optimized for dark backgrounds
- `assets/logo/bsd-lockup-horizontal.svg`: primary website lockup on light backgrounds
- `assets/logo/bsd-lockup-horizontal-dark.svg`: primary website lockup on dark backgrounds
- `assets/logo/bsd-lockup-horizontal-mono.svg`: monochrome horizontal lockup
- `assets/logo/bsd-lockup-stacked.svg`: stacked lockup for posters and square placements
- `assets/logo/bsd-lockup-stacked-dark.svg`: stacked lockup for dark sections
- `assets/logo/bsd-lockup-stacked-mono.svg`: monochrome stacked lockup

## Brand Rationale

The symbol now removes the earlier orbit dots and focuses on continuous, interlocking arcs. The result is calmer and more system-like:

- the outer arc suggests a global brain-state trajectory
- the counter arc implies return, feedback, and closed-loop control
- the pale inner arc hints at latent structure, manifold flow, and layered signals

The rounded-square container is kept as an app-icon option for favicon and website identity, while the raw symbol gives the lab a cleaner mark for editorial and research contexts.

## Visual System

- Primary palette: `#020817`, `#0F172A`, `#334155`, `#475569`, `#94A3B8`, `#CBD5E1`, `#F8FAFC`
- Style target: low ornament, flat surfaces, tight contrast, shadcn-like UI restraint
- Typography target in SVG: `Geist` first, then `SF Pro Display`, then system sans-serif fallback

## Usage Rules

- Minimum size for the raw symbol: `20px`
- Minimum size for the container mark: `24px`
- Minimum width for the horizontal lockup: `150px`
- Use dark variants directly on near-black or graphite website sections
- Use monochrome variants for print, slide footers, and paper headers
- Do not reintroduce dots, shadows, glossy gradients, or decorative glows
- Keep at least one stroke-width worth of clear space around the symbol

## Notes

- The SVG files remain the source of truth
- PNG exports are regenerated by `scripts/export-logo-pngs.ps1`
- Open `preview/logo-preview.html` to compare container and raw symbol versions across light and dark backgrounds
